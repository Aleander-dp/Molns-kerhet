# Molns-kerhet
Molnmiljöer och säkerhetsåtgärder i AWS

## filer
Diagram på molnmiljö, en rapport och presentation i två format (pptx, pdf)

## Säkerhetsåtgärder
- route 53
- WAF
- Application Load Balancer
- Endast ett publikt subnät med, där load balancer placeras
- Flera privata subnät per region & AZ
- Segmentering
- Uppdelning av interna resurser
- Private API Gateways
- Access Policies
- Security Groups
- IAM Roller baserade på behov
- Least Privilage
- CIA Baserat

## Syfte
Syftet med detta projekt var att bli bekväm med att skapa säkra molnmiljöer, konfigurera säkerhetsgrupper, skapa anpassade IAM roller och Skydda interna resurser på lämpligt sätt. Miljön skulle passa de behov ett fiktivt företag hade både tekniskt och ekonomiskt.
